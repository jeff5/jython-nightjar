Welcome to Jython 2.5.4rc1!
===========================

This is the first release candidate of Jython 2.5.4.

Thanks to Adconion Media Group (http://www.adconion.com/) for sponsoring this
release, and thanks to all who contribute to Jython.

This release fixes numerous bugs since the 2.5.3 release of Jython. Some
highlights include:

* The file upload fix for Tomcat and Jetty has been changed.
* Some NullPointerExceptions where fixed.

Please see the NEWS file for detailed release notes.  The release was compiled
on Ubuntu with JDK 7 and requires JDK 5 to run.

Please try this out and report any bugs at http://bugs.jython.org.
